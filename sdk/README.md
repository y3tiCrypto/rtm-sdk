## ⚡ Integration Overview

All package clients are structured to interface with the Raptoreum Core daemon JSON-RPC port (`8766` on mainnet, `18766` on testnet).

To configure your node and verify connection health, refer to the [docs/integration_guide.md](file:///e:/RTM-Scripts/rtm-sdk/docs/integration_guide.md) file in the root directory.
